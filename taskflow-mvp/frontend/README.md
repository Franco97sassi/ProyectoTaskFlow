# TaskFlow Frontend
Frontend Angular de TaskFlow MVP. Incluye login, rutas protegidas, dashboard con cards, listado filtrable de tareas y formulario de creación/edición con selector de proyecto y usuario asignado.
## Comandos
```bash
npm install
npm run start
npm run build
```

La app espera el backend en `http://localhost:8081` según `src/environments/environment.ts`.
Para la documentación completa del proyecto, ver [`../README.md`](../README.md).
